# Loan Recovery and Stress Testing Engine

An end-to-end credit risk analytics project built in a single Jupyter Notebook (`main.ipynb`), designed for Indian NBFC and fintech-style loan portfolios.  
The project simulates risk, stress scenarios, capital buffers, and time-based loss evolution with clear visualizations and business-ready insights.

---

## Overview

This project models the complete credit risk lifecycle:

- DPD classification (Standard, SMA-1, SMA-2, NPA)
- Expected Loss calculation using PD × LGD × EAD
- Portfolio-level stress testing (Base vs Severe)
- Capital buffer and threshold analysis
- Regression-based time stress simulation
- Management-focused risk summaries
- Advanced visualizations including thermometers and stress overlays

---

## Project Highlights

- Single-notebook design for easy review
- Explainable and transparent risk modeling
- Stress testing aligned with real NBFC practices
